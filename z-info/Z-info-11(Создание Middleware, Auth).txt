Проверка всех возможных команд создания: php artisan make: 
(получим полный список с кратким описанием всего, что можно создать)

Для проверки является ли пользователь администратором нам нужно создать соответствующий middleware:

	php artisan make:middleware CheckIsAdmin
	

Далее для созданного middleware нужно прописать путь к нему в файле: Kernel.php - в массив в свойстве: protected $routeMiddleware т.е. пропишем алиас

Затем этот алиас сможем мспользовать внутри роутинга
